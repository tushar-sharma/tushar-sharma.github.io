### Software Update

On Mac, to list all outdated softwares, easiest way is to open termainal run these commands

```bash 
# list all outdated softwares
softwareupdate --list

# update all outdated software
softwareupdate --install --all 

# update a particular software
softwareupdate --install <product>
```
