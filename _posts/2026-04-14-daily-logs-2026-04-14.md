# Claude code

Add this to `.claude/commands/import-cert.md` 

```
Import a certificate into JVM truststore

Ask me for : 
1. `alias` - a short identifier for this cert (e.g. nexus)
2. `cert_file` - path to the .crt or .pem file to import 

Then run 

sudo keytool --import -alias <alias> -file <cert_file>  -keystore $JAVA_HOME/lib/security/cacerts

Remind me to run `keytool --list -keystore $JAVA_HOME/lib/security/cacerts | grep <alias>' to confirm it was imported.
```

Restart claude and type `/import-cert`. 


Also add this to your `CLAUDE.md` workflows

```
## Available Workflows
```
