## Replace 32-bit Wine with 64-bit on Steam Deck

I was trying to install **Detroit: Become Human** and ran into an error:

    could not load kernel32.dll

To fix this, I replaced 32-bit Wine with 64-bit Wine.

### Step 1: Remove existing 32-bit Wine prefixes

``` bash
rm -rf ~/.wine
rm -rf ~/.local/share/wineprefixes
```

### Step 2: Install 64-bit Wine (default)

By default, this installs the 64-bit version of Wine:

``` bash
sudo pacman -S wine
```

### Step 3: Add Prefix with Winetricks

``` bash
WINETRICKS=/home/deck/.local/share/wineprefixes winetricks vcrun6sp6
```

⚠️ **Note:** Proton Experimental doesn't work with 32-bit Wine. Stick to
64-bit for better compatibility.
