## Function Overload in Python

Let's say we a simple class `Person` 


```python
data = {
    'age': 23,
    'salary': 10
}

class Person:
    def __init__(self, data) -> None:
        self.__dict__.update(data
        
person = Person(data)
print(person.age, person.salary)
```
