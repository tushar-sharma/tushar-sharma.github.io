## uv 

**uv** is the package manager for python. Just like poetry. It's written in rust. 

Installation 

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

Install dependencies

```bash
uv sync
```

## dexter

```bash
git clone https://github.com/virattt/dexter && cd $_
```
